CONTEXTO Y OBJETIVO
Desarrolla una aplicación web profesional para gimnasios que integre gestión de rutinas personalizadas y tienda virtual de suplementos/accesorios. La aplicación debe ser escalable, visualmente atractiva y funcional para múltiples sedes.

ARQUITECTURA TÉCNICA REQUERIDA
Backend - PHP MVC
* Framework: PHP 8.x con arquitectura MVC personalizada o framework como Laravel/CodeIgniter
* Base de datos: MySQL con diseño relacional optimizado
* Estructura MVC: 
  o Models: Gestión de datos (usuarios, rutinas, productos, sedes, instructores)
  o Views: Interfaces HTML/CSS/JS responsivas y profesionales
  o Controllers: Lógica de negocio y manejo de peticiones
* APIs RESTful para integración frontend-backend
* Sistema de autenticación robusto con roles (admin, instructor, cliente)

Frontend - Experiencia Visual Profesional
* HTML5 semántico con estructura optimizada
* CSS3 avanzado con: 
  o Animations y transitions suaves
  o Grid/Flexbox para layouts responsivos
  o Variables CSS para temas personalizables
  o Efectos hover y micro-interacciones
* JavaScript ES6+ con: 
  o Manipulación del DOM moderna
  o Fetch API para peticiones asíncronas
  o Efectos visuales y animaciones
  o Validación de formularios en tiempo real

FUNCIONALIDADES CORE
1. SISTEMA DE RUTINAS PERSONALIZADAS
Gestión de Rutinas
* Creador de rutinas con interfaz drag-and-drop
* Biblioteca de ejercicios categorizada (fuerza, cardio, flexibilidad, etc.)
* Configuración detallada por ejercicio: 
  o Nombre, descripción, músculos trabajados
  o Series, repeticiones, peso, descanso, tempo
  o Notas específicas del instructor
  o Nivel de dificultad y progresión

Sistema Multimedia
* Subida de videos optimizada con: 
  o Compresión automática para web
  o Múltiples resoluciones (360p, 720p, 1080p)
  o Miniaturas generadas automáticamente
  o Player HTML5 personalizado con controles
* Galería de imágenes para ejercicios
* Sistema de etiquetado para búsqueda avanzada

Asignación y Seguimiento
* Asignación de rutinas a clientes específicos
* Calendario de entrenamientos interactivo
* Tracking de progreso con gráficos y métricas
* Historial completo de entrenamientos realizados

2. TIENDA VIRTUAL PROFESIONAL
Catálogo de Productos
* Categorización avanzada: 
  o Suplementos (proteínas, pre-entrenos, vitaminas, etc.)
  o Accesorios (ropa, equipos, gadgets)
  o Subcategorías dinámicas
* Ficha de producto completa:

Fase 4: Integración y Clases Grupales (4 semanas)

TECNOLOGÍAS RECOMENDADAS
* Backend: PHP 8.1+, MySQL 8.0+
* Frontend: HTML5, CSS3, JavaScript ES6+
* Librerías JS: Alpine.js, Chart.js, Swiper
* Procesadores CSS: SASS/SCSS
* Herramientas de build: Webpack/Vite
* Versionado: Git con flujo GitFlow
* CI/CD: GitHub Actions o similar
* Hosting: VPS optimizado con Nginx

INSTRUCCIÓN FINAL: Desarrolla esta aplicación priorizando la experiencia del usuario, la escalabilidad y la integración perfecta entre todos los módulos. Cada funcionalidad debe sentirse como parte de un ecosistema cohesivo, no como módulos independientes. Implementa un diseño visualmente impactante con la paleta de colores corporativa (naranja #FF6B00 como color principal) y asegura que la navegación sea intuitiva y fluida en todos los dispositivos.
